# Erika AI YouTube Assistant

AI-powered YouTube brand building assistant for Erika Vieira. Built on Cloudflare Workers.

## Project Structure

- `src/` - Application source code
- `public/` - Static assets served to the browser
- `public/assets/` - Images, fonts, and other static files
- `migrations/` - Database migration files

## Development

```bash
npm run dev      # Start local dev server
npm run build    # Dry-run deploy (validate)
npm run deploy   # Deploy to Cloudflare Workers
```
